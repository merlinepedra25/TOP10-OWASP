你能如何幫助

在目前階段，我們需要

  • 資料科學家 - 協助確認我們的資料分析結果
  • 網頁設計師 - 我們需要做一個對行動裝置友善的版本
  • 翻譯人員 - 請協助確認英文版本文字並確認可被翻譯到其他語言版本
  • ASVS, Testing Guide, 和 Code Review Guide 專案領袖 - 請使用我們的資料並協助我們把其他的文件和標準連接起來
